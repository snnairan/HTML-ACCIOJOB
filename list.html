Block Elements

Block elements are those which starts from a new line. Block elements take complete width of the screen.


    Example:

  1. header
  2. footer
  3. div
  4. main
  5. nav
  6. article
  7. aside
  8. section


Inline Elements

Inline elements starts from the point where the previous element ended.


    Example:

  • span
  • script
  • select
  • img
  • map
  • object
  • br
  • label
